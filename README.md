# Links Junkie

Copy custom link text (ex. PlainText, HTML, TiddlyWiki), from shortcut key and context menu, for Google Chrome browser.

### Features
- copy selected text.
- Keyboard shortcuts
- Flexible context menus
- URL shortening
- Notifications
